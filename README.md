# 医院商城管理后台

## 项目概述

医院商城管理后台是医疗机构数字化转型的关键组成部分，旨在通过线上平台实现药品、医疗器械、保健品等医疗相关商品的管理与销售，提升医院服务效率，优化患者体验。

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **图表库**: ECharts
- **HTTP请求**: Axios
- **日期处理**: Day.js

## 功能模块

### 1. 用户管理
- 用户列表查看、搜索、筛选
- 用户信息编辑
- 用户状态管理（冻结/解冻/封禁）
- 会员等级管理
- 实名认证审核

### 2. 商品管理
- 商品列表查看、搜索、筛选
- 商品新增、编辑、删除
- 商品上下架管理
- 商品分类管理（支持多级分类）
- 效期管理、库存预警
- 批量导入导出

### 3. 订单管理
- 订单列表查看、搜索、筛选
- 订单状态管理
- 订单详情查看
- 订单发货处理
- 退款管理

### 4. 库存管理
- 实时库存查看
- 入库/出库管理
- 库存预警
- 库存调拨
- 库存记录查询

### 5. 处方管理
- 处方接收与审核
- 处方详情查看
- 审核结果记录
- 处方统计

### 6. 配送管理
- 配送订单管理
- 配送状态跟踪
- 物流信息对接
- 自提管理

### 7. 会员与营销
- 会员等级管理
- 积分管理
- 优惠券管理
- 促销活动管理

### 8. 财务管理
- 交易记录查看
- 对账管理
- 退款管理
- 财务报表

### 9. 数据统计
- 销售趋势分析
- 商品销量排行
- 用户增长趋势
- 订单状态分布
- 数据可视化图表

### 10. 系统设置
- 基本信息设置
- 管理员账号管理
- 角色权限管理
- 安全设置

## 安装与运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run dev
```

### 生产环境构建
```bash
npm run build
```

### 代码检查
```bash
npm run lint
```
